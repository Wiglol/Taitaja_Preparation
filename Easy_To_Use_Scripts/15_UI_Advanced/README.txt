15_UI_Advanced (Unity 2022 LTS)
------------------------------
UI scripts for common judging points.

Scripts:
- HealthBarSlider: connects Health to a UI Slider
- ObjectiveTextUI: set objective text from code
- GameOverScreen: shows lose panel when health reaches 0
- WinScreen: shows win panel when GameManager win state happens
- ScorePopupText: floating score popup (simple)

Note:
- Slider requires UnityEngine.UI.
