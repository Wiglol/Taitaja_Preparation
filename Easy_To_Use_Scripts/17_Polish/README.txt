17_Polish (Unity 2022 LTS)
--------------------------
Small polish scripts that make prototypes feel finished fast.

Scripts:
- SpriteBob: gentle up/down motion (collectibles, UI icons)
- ScalePulse: pulsing scale (buttons/pickups)
- FadeSprite: fade out a SpriteRenderer over time
- ScreenFlash: flash a fullscreen UI Image (damage feedback)
